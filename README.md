## 📋 פורמט קובץ Excel

המערכת מזהה את העמודות **אוטומטית** לפי שמות העברי/אנגלי:

| עמודה | שמות שמזוהים |
|-------|-------------|
| **שם לקוח** (חובה) | שם, לקוח, name, customer |
| **כתובת** | כתובת, address, רחוב |
| **עגלות** | עגלות, עגלה, כמות, carts, qty |
| **החל משעה** | החל משעה, משעה, time_from, from |
| **עד שעה** | עד שעה, לשעה, time_to, until |
| **הערות** | הערות, הערה, notes |

**כל שאר המידע** (כתובת, שעות, הערות) נלקח **תמיד מהקובץ היומי** — כלום לא נשמר בין ימים.

| כלל | פירוט |
|-----|-------|
| 🏠 בסיס | כל קו יוצא וחוזר למושב חגלה (32.42°N, 35.50°E) |

| ↕️ כיוון | **לא מערבבים** לקוחות צפון מחגלה עם דרום |

| 📍 סדר | **Nearest Neighbor** — כל עצירה קרובה לקודמת |

```
app/

  geocode.ts  ← המרת כתובות
  routing.ts  ← אלגוריתם חלוקה לקווים
```
